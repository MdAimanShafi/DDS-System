🔌 DDS Extension Setup

Connect the Panic Button extension to your dashboard

Step 1: Install Extension

Open chrome://extensions/ → Enable Developer Mode → Load Unpacked → Select panic-extension folder

Step 2: Copy Extension ID

After loading, Chrome will show an ID like abcdefghijklmnopqrstuvwxyz123456 under the extension name. Copy it.

Step 3: Paste ID Below

Paste the extension ID here and click Save:

✅ Extension connected! Panic button will now clear all cookies & logout from all websites.
